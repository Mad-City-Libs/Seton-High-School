**🎉 Seton Dance – “This or That?” Game Overview

This is a lightweight, web-based “This or That?” game designed to add optional, interactive fun around a school dance event.

The game presents students with quick either/or choices and generates a lighthearted result at the end. It is designed to be:

Fast

Mobile-friendly

Easy to share

Completely optional

Privacy-safe

**Purpose

The goal of this project is to:

Increase excitement leading up to a school dance

Provide a low-pressure social activity

Encourage peer-to-peer engagement

Demonstrate creative student-led web development

This is not an official school tool. It is a standalone student project built for fun.

**Features**

6-round “This or That?” format

Instant result screen

No login required

No personal data collected

Mobile responsive design

Simple, clean UI

Privacy & Data Policy

**This game:**

Does not collect personal information

Does not store user responses

Does not use tracking cookies

Does not require accounts or authentication

All interaction happens locally in the browser.

**Technical Details**

Built using HTML, CSS, and JavaScript

Designed for static hosting (e.g., GitHub Pages)

Optimized for quick loading and phone usability

**Project Philosophy**

This project reflects an interest in:

Interactive web design

Lightweight digital experiences

Community-based engagement

Safe and responsible student development
